# Chapter 3: A Synthetic Lab for Real Pharma Questions

Welcome to Chapter 3! This chapter teaches you to work with pharmaceutical data by building and analyzing a transparent synthetic dataset.

## What You'll Learn

In this chapter, you'll discover:
- **The pharmaceutical data landscape**: Claims, EHR, formulary, CRM, and public data sources
- **Data maturity and receipt lags**: Why recent claims are incomplete and how to handle them
- **Patient journeys across systems**: Following one patient from clinical order through pharmacy fill
- **Data quality auditing**: Identifying what each table really counts and what it misses
- **Synthetic data generation**: Building realistic pharmaceutical datasets with known structure and defects
