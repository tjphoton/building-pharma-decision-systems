# Chapter 4: Market Sizing and Patient Populations

Welcome to Chapter 4! This chapter teaches you how to build market size estimates from real-world data and understand the different ways to define a market.

## What You'll Learn

In this chapter, you'll discover:
- **Four market size definitions** - Understand the gap between true disease, coded diagnosis, label eligibility, and treated patients
- **From epidemiology to claims** - Connect true disease prevalence to coded diagnosis and paid claims
- **National calibration** - Use population anchors to adjust synthetic data for real-world comparison
- **Opportunity funnel** - Build step-by-step filters through diagnosis, eligibility, access probabilities, and treatment starts
- **Capture-recapture** - Estimate patients who fall through both source systems and don't appear in claims
- **Patient finding** - Use machine learning to rank undiagnosed patients by probability of the true condition
